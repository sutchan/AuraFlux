# Aura Flux 🎵👁️

### AI 驱动的 3D 音乐可视化与通感分析引擎 (v1.8.66)

[在线演示](https://aura.tanox.net) | [English](./README.md)

Aura Flux 是一款专业级的 Web 应用程序，可将音频频率转化为高保真生成艺术。基于 **React 18**、**Three.js** 和 **Google Gemini 3 AI** 构建，实现了极致性能与人工智能的完美融合。

## 🚀 核心特性

- **混合渲染管线：** 在高保真 WebGL 3D 场景 (R3F) 与经过优化的离屏 2D Canvas 渲染器之间无缝切换。
- **AI 通感 HUD：** 由 Gemini 3.0 Flash 驱动的实时歌曲识别、情绪分析和动态歌词显示。
- **4K 工作室录制：** 支持以高分辨率（最高 4K）导出您的视觉创作，并支持自定义码率和混音。
- **Bento 界面布局：** 为专业 VJ 设计的模块化、响应式控制系统。
- **本地隐私优先：** 所有频谱分析和图像渲染均在您的设备本地完成。

## 🎭 应用场景

### 1. 现场 VJ 与舞台设计
专为现场 DJ 表演、音乐会或派对环境设计。
- **配置：** 将笔记本电脑连接到大型 LED 墙或投影仪。
- **工作流：** 选用“动感光墙 (Kinetic Wall)”或“数位网格 (Digital Grid)”以获得强力的节奏冲击。利用快捷键（1-6 切换面板，R 随机化）在演出中实时调整视觉氛围，无需繁琐的操作。
- **优势：** 毫秒级音频响应，视觉动态与节奏严丝合缝。

### 2. 短视频创作与自媒体 (TikTok/B站/视频号)
在几分钟内创作出极具视觉冲击力的音乐宣发视频。
- **配置：** 在**工作室 (Studio)** 面板中切换至 9:16 (竖屏) 比例。
- **工作流：** 开启 **AI 歌词图层** 进行 4K 录制。歌词将根据音乐能量实时脉动和发光，为您提供通常需要数小时后期制作才能达到的动态图形效果。
- **优势：** AI 通感功能会自动分析曲风，并为您推荐最匹配的视觉调色盘。

### 3. 家居氛围与智能显示
将闲置的平板电脑或智能镜转变为一件活的数字艺术品。
- **配置：** 在浏览器中打开，切换全屏 (`F`)，并开启**自动轮播**模式。
- **工作流：** 配合 Lo-Fi 或环境音乐播放。使用“流光绸缎 (Silk Wave)”或“深空星云 (Nebula)”模式，营造一种宁静、变幻莫测的家居氛围。
- **优势：** PWA 应用支持离线运行，在“节能”画质下可长时间保持流畅显示。

### 4. 冥想放松与专注空间
将生成式艺术作为呼吸练习、专注力训练或冥想的视觉中心。
- **配置：** 选择“立体海浪 (Ocean Wave)”或“极光流 (Fluid Curves)”，配合柔和的单色调色盘。
- **工作流：** 开启**自定义图层文字**功能，显示冥想词或呼吸指令。文字的律动脉动可作为吸气与呼气的视觉引导。
- **优势：** 通过“视听共感”增强沉浸感，帮助用户更快速进入心流状态。

### 5. 专业音频频谱分析
音乐制作人或音响发烧友可视化音频频率分布的得力工具。
- **配置：** 在专业 UI 模式下选择“频谱仪 (Bars)”或“数位网格 (Digital Grid)”。
- **工作流：** 调节 FFT 采样大小以观察精细的谐波分布。高保真的频谱映射有助于实时识别频率峰值和声场宽度。

## 🛠️ 技术栈

- **核心框架：** React 18, TypeScript 5.7, Vite 6
- **图形渲染：** Three.js, React Three Fiber (R3F), 后处理滤镜
- **人工智能：** Google GenAI SDK (Gemini 3 Flash, Gemini 2.5 Flash Image)
- **音频处理：** Web Audio API, 使用 `jsmediatags` 解析 ID3 元数据

## 🎹 快捷键指南 (VJ 控制)

| 按键 | 动作 |
| :--- | :--- |
| `空格` | 切换麦克风 / 播放状态 |
| `F` | 切换全屏 |
| `L` | 开启/关闭 AI 歌词与通感 HUD |
| `H` | 隐藏/显示 界面 UI |
| `R` | 随机化美学方案 |
| `← →` | 循环切换视觉引擎 |
| `[ ]` | 循环切换色彩主题 |
| `1 - 6` | 切换设置面板 |
